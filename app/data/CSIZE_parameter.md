## CSIZE parameter

**Pages in table C**

**Summary**

* **Default value:** 1
* **Parameter type:** Tables
* **Where set:** During file creation
* **Related products:** All
* **Introduced:** Model 204 V6.1 or earlier

**Description**

The number of pages in Table C.  CSIZE has a limit of 16777216.
